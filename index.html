What is React Native ?

It is a framework that allows you to create applications for different platforms using the same codebase.

With React Native, companies can only create code once and use it to power their iOS and Android apps.

React Native was built on top of React, a JavaScript library that was already extremely popular when the mobile platform was released.

History

In 2012, Mark Zuckerberg admitted that “the biggest mistake we made as a company was that we bet too much on HTML. in 2013, Facebook developer Jordan Walk made a discovery,when he found a method to create UI elements for iOS apps using JavaScript.

This caused a sensation and a special hackathon was organized to explore how much mobile development can be done using traditional JavaScript web-solutions.

React vs React Native

Both React and React Native use a mix of JavaScript and a special markup language called JSX.

However, the syntax used to render elements in JSX components is different between React and React Native. React JSX renders HTML-like components such as <h1> , <p> , etc. Meanwhile, react-native renders the application's native view components such as <View> , <Text> , <Image>, <ScrollView>

Code for example 


          (
          import React, {useState} from 'react';
          import {View, Text, Button, StyleSheet} from 'react-native';

          const App = () => {

              <View> style={styles.container}/>

          // React Native Styles

            },
          }))

How does react native work

React Native uses what is called a “bridge”. JavaScript stream and native streams are written in completely different languages, but with the bridging feature system makes bidirectional communication possible.

Cross-platform development

Cross-platform development is the practice of creating software that is compatible with more than one type of hardware platform.

Logical components will be the same for android and iOS

  • Large audience
  • Platform сonsistency
  • Reusable code
  • Cost reduction

Why React Native is unique?

The difference between React Native and other cross-platform development solutions (such as Flutter, Cordova and PhoneGap) is that React Native doesn't render a WebView in code. It runs on real, native views and components. This is one of the reasons for React Native's impressive success.
